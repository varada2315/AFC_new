# 🏦 Replit Prompt: Ashish Finance Services Website (v2)

Build a **premium 5-page financial services website** for **Ashish Finance Services** by Mr. Ashish Vilas Urkude.

## TECH STACK

- **HTML5 + CSS3 + Vanilla JavaScript** — no frameworks
- Separate HTML files per page, shared `style.css` and `script.js`
- Google Fonts
- Fully mobile responsive
- No backend — static site only

- **Logo / Brand Name**: `Cinzel` or `Playfair Display` — serif, gold
- **Headings**: `Montserrat` Bold — uppercase for section titles
- **Body**: `DM Sans` — clean and readable
- **Marathi text**: `Noto Sans Devanagari`
- Load all from Google Fonts

## ANIMATIONS & JS BEHAVIORS

1. **Scroll fade-in**: `IntersectionObserver` — sections slide up + fade in when entering viewport
2. **Counter animation**: `1000+` and `500+` count up with easing when stats section enters viewport
3. **Navbar scroll**: Adds `box-shadow` and slight size reduction after 80px scroll
4. **Lightbox**: Certificate image modal open/close with keyboard `Esc` support
5. **Mobile menu**: Smooth slide-down animation
6. **Hover on service cards**: `translateY(-6px)` + gold glow `box-shadow`
7. **Hover on social cards**: Slight scale `1.03`
